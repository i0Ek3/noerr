# noerr

> You will never see &#34;if err != nil&#34; in the go file again!

## Getting Started

### Install

```sh
go get https://github.com/i0Ek3/noerr
```

### Usage

Please follows above step in your Go project and then run the command below:

```sh
noerr.NoErr(err)
```
